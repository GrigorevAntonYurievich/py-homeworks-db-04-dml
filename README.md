ISPID - ID исполнителя
NAMEISP - название исполнителя

NAMEALB - имя альбома

DLITEL - длительность трека
SBORNIK_ID - id сборника
SBORNIK_NAME - имя сборника
DATA_RELIS - дата релиза сборника
